# @muninhq/config-generic-baseline

The neutral, vertical-agnostic default configuration cartridge for Munin
(`cartridgePackage('generic-baseline')`). Provides baseline entity-type schemas,
terminology, and retrieval defaults so a local Munin instance has a sensible
generic configuration out of the box. Depends only on `@muninhq/shared`.
