# Project Atlas — Project Brief

Atlas is the migration of our legacy reporting stack onto the new data warehouse. The project is led by Sarah Chen, our data engineering lead. It began on 1 March 2026 with a planned six-month delivery window and is currently active.

Priya Anand and Marcus Webb are the two engineers working on Atlas alongside Sarah. Priya owns the data-modelling workstream; Marcus is responsible for the extraction connectors.

The goal is to retire the legacy reporting database entirely by the end of the financial year, with no loss of historical report fidelity.
