# Project Cobalt — Project Brief (Restricted)

Cobalt is our security-compliance remediation project, addressing the findings from the external penetration test. It is managed by Lena Fischer, our security engineer, and started on 5 January 2026. The project is active and treated as confidential.

Cobalt covers closing the high-severity findings, rolling out mandatory multi-factor authentication, and preparing the evidence pack for the Cyber Essentials Plus assessment.
